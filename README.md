# 三国志英杰传编辑器

## 启动后端

```
python3 backend/reko3data.py
```
默认监听localhost:8888

## 启动前端

```
yarn start
```
默认监听localhost:8000

## 其他问题

* LS11解析程序在backend/ls11
* 所需修改的文件放在backend/reko3下面，已用LS11解压
* react加载组件太慢，序章就花了一分多钟，其他的更可怕
